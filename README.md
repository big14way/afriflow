### Installation

```bash
# Clone the repository

# Install all dependencies
npm run install:all

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Deploy contracts to Cronos Testnet
npm run deploy:testnet

# Start the development servers
npm run dev
```
